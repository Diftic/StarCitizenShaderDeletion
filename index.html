SC Performance Tool

A guided maintenance wizard for Star Citizen — scan, optimise, and clean shader caches in four steps.

Download ShaderCacheNuke.exe

Windows 10 / 11  ·  ~10 MB  ·  Requests admin automatically  ·  v3.0.0

Features

🔍

System Analysis

Scans shader caches, GPU caches, temp files, HAGS, power plan, and running processes.

🎯

Star Citizen Caches

Detects all SC shader folders across every drive and channel — LIVE, PTU, EPTU, TECH-PREVIEW.

🟢

GPU Driver Caches

NVIDIA DXCache, GLCache, ComputeCache, NV_Cache. AMD DxCache, GLCache, VkCache, Dx9Cache. DirectX D3DSCache.

⚙️

System Checks

HAGS and power plan status with one-click links to the right settings page.

🔒

Locked File Handling

Files in use are scheduled for deletion at next boot — nothing gets silently skipped.

🧠

Standby Memory Clear

Purges the Windows standby list to reclaim memory before a play session.

🌙

Light / Dark Theme

Toggle between themes from the toolbar at any time.

🔄

Auto-Update Check

Notifies you when a new release is available with a direct link to download it.

How It Works

1

Analysis

The tool scans your system and produces a full report — cache sizes, running processes, HAGS and power plan status, available RAM. Nothing is changed at this step.

2

Manual Actions

Flagged items that need manual attention are listed with status indicators and direct-open buttons. HAGS disabled? Opens Graphics Settings. Wrong power plan? Opens Power Options.

3

Automated Cleaning

A checklist of everything that can be cleared automatically, pre-ticked based on what was found. Review, adjust, and click Run Cleaning.

4

Done

Full per-operation summary with pass/fail results. If locked files were scheduled for reboot deletion, a restart prompt is shown.

When to Use

  • After Star Citizen patches or updates
  • When experiencing graphical glitches, stuttering, or shader-related issues
  • After GPU driver updates
  • Before a play session if performance has degraded
  • When troubleshooting unexplained FPS drops or hitching

Run from Source

git clone https://github.com/Diftic/StarCitizenShaderDeletion.git
cd StarCitizenShaderDeletion
python nuke_shaders_gui.py

Requires Python 3.10+ on Windows 10 / 11.

Notes

  • The first SC launch after clearing shader caches will take longer as shaders recompile — this is normal.
  • Standby memory clear requires administrator privileges.
  • The tool only modifies files within known cache locations.
